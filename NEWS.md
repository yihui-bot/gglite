# gglite 0.1

- Added two package vignettes (using `litedown::vignette` engine):
  "Introduction to gglite" (an overview of the Grammar of Graphics components
  with examples) and "gglite for ggplot2 Users" (a comparison guide for ggplot2
  users).

- A lightweight R interface to the
  [AntV G2](https://g2.antv.antgroup.com/) JavaScript visualization library
  with a ggplot2-style API. Create interactive charts using the Grammar of
  Graphics with support for 35+ geometry types (marks), scales, coordinates,
  themes, transforms, facets, animations, and chart components (axes, legends,
  titles, tooltips, labels, sliders, and scrollbars). Renders in R Markdown,
  litedown, Shiny, and standalone HTML previews.
